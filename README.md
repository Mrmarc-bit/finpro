# 🚀 Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

---

## 2. Open Project

Use:
- VS Code
- Live Server Extension

DO NOT open using:

```bash
file://
```

Because browsers dislike local fetch requests with the passion of a thousand security engineers.

---

## 3. Run Project

Open with Live Server:

```bash
http://127.0.0.1:5500
```
